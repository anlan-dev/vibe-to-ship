# Contributing to AI Co-Founder

Thanks for your interest in contributing! This project is open to everyone.

## Ways to contribute

### 1. Add a case study

Built something with the AI Co-Founder prompt? Share it!

1. Fork this repo
2. Add your product to `examples/your-product-name/`
3. Update `examples/CASES.md` with your case study
4. Submit a PR

### 2. Improve the prompt

The prompt in `prompts/cofounder.md` is the core of this project. If you have ideas to make it better:

1. Fork this repo
2. Edit `prompts/cofounder.md`
3. Describe what you changed and why in the PR description
4. Submit a PR

### 3. Report bugs

Found a problem? Open an issue with:

- What you were trying to do
- What happened instead
- Steps to reproduce

### 4. Suggest features

Have an idea? Open an issue with:

- What you want to do
- Why it would be useful
- Any examples you've seen elsewhere

## Guidelines

- Be respectful. See [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md).
- Keep PRs focused. One change per PR.
- Write clear commit messages.
- Test your changes before submitting.

## Questions?

Open an issue with the "question" label.
